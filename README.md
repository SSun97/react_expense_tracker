# This is a web app for a expense tracker
Deployed on: https://ssun97.github.io/react_expense_tracker/

## How to use this repo
1. Install node.js
2. Install React
3. run the commands below 
``` 
$ git clone https://github.com/SSun97/react_expense_tracker.git
$ git checkout main
$ npm install
$ npm start
```
